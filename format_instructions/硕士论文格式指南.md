# 北京林业大学硕士学位论文格式指南

> 基于《研究生学位论文写作指南（2023版）》整理

---

## 一、总体要求

### 1.1 基本要求

**学术型硕士学位论文基本要求：**

1. 论文选题围绕国家发展重大战略需求，具有学术创新性或应用指导意义
2. 体现对本学科坚实的理论基础和系统专业知识的掌握
3. 掌握研究领域先进的分析理论与方法
4. 论文各单元具有系统完整的科学逻辑联系
5. 试验数据来源真实可靠，分析科学，语言简练，图表清晰
6. 论文应在导师指导下由硕士生独立完成
7. 能综合体现作者具备独立从事科研工作能力

**论文工作量要求：**
- 用于论文工作的时间应不少于一年半
- 论文字数原则上在**3万字以上**

### 1.2 语言与表述
~
- 除留学生外，学位论文**用汉语书写**
- 使用规范的简化汉字（引用外文文献除外）
- 专用名词、术语可采用国际通用代号
- 量及其单位符号应符合国家标准 GB3100-1993、GB3101-1993
- 图表中的描述性词组或语句须使用汉语

### 1.3 页面格式

| 项目 | 规格 |
|------|------|
| 纸张 | A4 |
| 上边距 | 3cm |
| 下边距 | 2.5cm |
| 左边距 | 3cm |
| 右边距 | 2.5cm |
| 页眉 | 五号宋体，边距2cm |
| 页眉内容 | 偶数页：论文题目；奇数页：所在章节题目 |

**页码格式：**
- 中文摘要至目录（或主要符号对照表）：罗马数字 Ⅰ、Ⅱ、Ⅲ...
- 正文开始至论文结束：阿拉伯数字 1、2、3...
- 页码位置：页面下部居中，Times New Roman 五号字体
- 页码两侧不加"-"或"---"等修饰线

---

## 二、论文结构与装订顺序

### 2.1 必需组成部分（按顺序）

1. **封面** ← 单面印刷，不加页码
2. **独创性声明和使用授权声明** ← 单面印刷，不加页码
3. **答辩委员会成员信息** ← 单面印刷，不加页码
4. **中文摘要** ← 双面印刷开始，罗马数字页码开始
5. **英文摘要（Abstract）**
6. **目录**
7. **主要符号对照表**（需要时）
8. **正文**（引言、各章内容）← 阿拉伯数字页码开始
9. **结论**
10. **参考文献**
11. **附录**（需要时）
12. **个人简介**
13. **导师简介**
14. **获得成果目录清单**
15. **致谢**

### 2.2 各部分分页要求

- 各部分各自独立成为一部分
- 每部分从新的一页开始
- 按顺序装订

## 三、各部分格式详细规范

### 3.1 封面

- **论文题目**：不超过25字，可分两行排列，中英文对照
- **作者及导师信息**：以研究生信息管理系统为准
- **导师信息**：一般只写一名，副导师需经批准并备案后并列
- **职称标注**：教授、副教授、研究员等

**封面颜色（纸质版）：**
- 博士：淡黄色
- 学术型硕士：淡绿色
- 专业学位硕士：淡蓝色

### 3.2 独创性声明和使用授权声明

- 位于论文第一页
- 独创性声明需作者本人签署姓名
- 版权使用授权声明需作者和导师签署姓名

### 3.3 答辩委员会成员信息

- 位于论文第二页
- 根据实际情况如实填写

### 3.4 中文摘要

**内容要求（硕士500字左右）：**
- 选题背景
- 研究目的
- 基本研究方法
- 主要研究结果
- 论文创新贡献及其理论与实际意义

**格式要求：**
| 元素 | 格式 |
|------|------|
| 标题"摘要" | 三号黑体，单倍行距，段前0.5行，段后1行 |
| 摘要正文 | 小四号宋体，1.5倍行距 |
| 关键词标题 | 小四号宋体 |
| 关键词 | 3～5个，逗号间隔 |

### 3.5 英文摘要（Abstract）

**内容结构：**
1. 英文题目
2. 研究生姓名（格式："Master Candidate..."）
3. 专业名称（括弧内）
4. 导师姓名（格式："Directed by..."）
5. Abstract 正文
6. Key Words

**格式要求：**
| 元素 | 格式 |
|------|------|
| 题目 | 小三号 Times New Roman，单倍行距，段前0.5行，段后1行 |
| 摘要正文 | 五号 Times New Roman，1.5倍行距 |
| Key Words | 小四号 Times New Roman |

### 3.6 目录

**内容要求：**
- 从引言开始，目录之前的内容及目录本身不列入
- 只列出论文的**三级结构标题**
- 次级标题依次缩进1格

**格式要求：**
| 元素 | 格式 |
|------|------|
| 标题"目录" | 三号宋体，居中，段前段后0.5行距 |
| 目录内容 | 五号宋体，全部左对齐，段前段后为0，行距1.25行 |

### 3.7 主要符号对照表（如有）

- 当论文使用大量物理量符号、标志、缩略词等时设置
- 如数量不多，可在论文中出现时随即说明

### 3.8 正文

#### 3.8.1 论文结构框架（自然科学/实验科学适用）

```
引言
    研究主题说明，选题背景，目的意义

1 论文主题领域研究进展与趋势
    (1) 研究领域发展概述
    (2) 研究重点问题与内容述评
    (3) 研究理论与方法述评
    (4) 主要研究结论述评
    (5) 研究局限与待解决问题述评
    (6) 领域研究发展趋势
    (7) 研究对象本主题相关研究进展与待解决问题
    (8) 论文立题依据、研究假设与任务

2 材料与方法
    研究思路、研究对象、实验方案、数据收集方法、统计分析方法

3 结果和分析
    (1) 取得的研究结果
    (2) 对结果的分析解释
    (3) 概括（主要科学发现）

4 讨论与展望

5 结论

参考文献
附录
```

#### 3.8.2 标题格式

**主题结构标题（三级）：**

| 级别 | 序号格式 | 字体格式 |
|------|----------|----------|
| 一级标题 | 1 | 黑体三号，段前0.5行，段后0.5行 |
| 二级标题 | 1.1 | 黑体小三号，段前0.5行，段后0行 |
| 三级标题 | 1.1.1 | 宋体四号 |

**条目标题（三级以下）：**

| 级别 | 序号格式 |
|------|----------|
| 第1级 | ⑴ |
| 第2级 | ① |
| 第3级 | a. b. c. d... |

**条目标题字体：** 宋体四号

#### 3.8.3 正文格式

- 字体：宋体小四号
- 段落：首行缩进
- 行距：1.25行距

#### 3.8.4 图表格式

**图的要求：**
- 图与正文之间要有一行的间距
- 图序及图名置于图的**下方**
- 图题目中文文字有英文标注
- 中文：五号宋体
- 英文：五号 Times New Roman
- 图采用阿拉伯数字分级编写（如图3.1）

**表的要求：**
- 表与正文之间要有一行的间距
- 表序及表名置于表的**上方**
- 表题目中文文字有英文标注
- 中文：五号宋体
- 英文：五号 Times New Roman
- 表采用阿拉伯数字分级编写（如表3.1）
- 表格使用**三线表**
- 表头文字：宋体加粗
- 表内文字：宋体五号

**图表附注：**
- 写在图或表的下方
- 采用英文小写字母顺序编号（如"注a:"）

**图表示例：**
```
表3.1 愈伤组织诱导试验接种5个山杨无性系的不同外植体类型
Table 3.1 Type of explant on callus induction of 5 aspen clones

图3.1 白杨无性系1/2MS20+IBA0.5mg/L20d生根情况
Figure 3.1 Effect of genotype on rooting after 20 days cultured on 1/2MS20+IBA0.5mg/L
```

#### 3.8.5 公式格式

- 字体：小四号宋体
- 编号：阿拉伯数字分级编写，用括号括起写在右边行末
- 其间不加虚线

**示例：**
```
E＝Vρ/（M-M₁-M₂）                           （3-1）
```

### 3.9 讨论（与展望）

**内容要求：**
1. 给出由研究结果揭示的客观原理、因果联系及其普遍意义
2. 分析阐述研究是否达到原定目标
3. 与他人相似研究相比较，讨论异同点
4. 对新结果、现象进行进一步的分析解释
5. 根据研究结果进行推论，提出经验模型或科学假设
6. 论文的不足之处、局限和进一步的研究展望

**写作要求：**
- 要逻辑严密、证据充分、论证周严
- 讨论要有论题来说明要阐述观点的来源

### 3.10 结论

**内容要求：**
1. 论文结果说明了什么问题
2. 得出了什么规律性的东西
3. 解决了什么理论和实际问题
4. 对前人研究结果做了哪些修正、补充、拓展、发展、证实或否定
5. 提出了哪些新的学术观点和建议

**写作要求：**
- 突出重点，集中表述经分析、论证、提炼、归纳后的总观点
- 不是简单的结果重复
- 撰写要准确、完整、明确、精炼、实事求是

### 3.11 参考文献

**格式要求：**
| 元素 | 格式 |
|------|------|
| 标题 | 三号黑体，段前0.5行，段后0.5行 |
| 正文 | 宋体五号，1.25倍行距 |

**引用格式：**
- 使用数字序号形式：[1]、[2]、[3]...
- 数字及中括号采用**上角标**
- 与参考文献编号一一对应

**引用规则：**

1. **句首点名作者引用：**
   - 单作者：XXX[21]指出，......
   - 两个作者：Tim和Adams[18]
   - 多作者：丁一等[1]；Tim等[2]

2. **句末引用：**
   - 阐述与概述内容......[2,5]

3. **连续引用多篇：**
   - 序号相连且≥3个：[5-9]
   - 序号不相连：[6,9,12-15]

### 3.12 附录（如有）

- 格式与正文相同
- 用大写字母编序号：附录A、附录B、附录C...
- 只有一个附录时也要编序号：附录A
- 每个附录应有标题，序号与标题之间空一个汉字符

### 3.13 个人简介

**内容包括：**
- 姓名、性别、出生年月、籍贯
- 本科及研究生所在学校专业等情况
- 获奖及发表论文情况
- 工作情况或工作去向

**格式要求：**
| 元素 | 格式 |
|------|------|
| 标题 | 三号黑体，段前0.5行，段后0.5行 |
| 正文 | 宋体小四号，首行缩进，1.25行距 |

### 3.14 导师简介

**内容包括：**
- 姓名、性别、出生年月、籍贯、职称
- 社会职务、学术研究情况
- 获奖及发表论文情况
- 研究生工作情况

**格式要求：** 同个人简介

**注意：** 个人简介和导师简介分页单列

### 3.15 获得成果目录清单

**分三部分按顺序列出：**

1. **已经刊载**
   - 按参考文献格式书写
   - 括号内注明：检索类型、检索号、影响因子
   - 未被检索的注明期刊级别（SCI源刊、EI源刊等）

2. **已录用未刊载**
   - 按参考文献格式书写
   - 括号内注明已被XX期刊录用及期刊级别

3. **其他研究成果**
   - 研究项目、专利、获奖等
   - 专利：专利名称、作者、专利号
   - 奖项：获奖名称、颁奖部门、获奖时间及名次

**格式要求：**
| 元素 | 格式 |
|------|------|
| 标题 | 三号黑体，段前0.5行，段后0.5行 |
| 正文 | 宋体小四号，首行缩进，1.25行距 |

### 3.16 致谢

**内容包括：**
- 国家科学基金、奖学金基金、合同单位、资助企业
- 协助完成研究工作和提供便利条件的组织或个人
- 提出建议和提供帮助的人
- 转载和引用权的资料所有者
- 其他应感谢的组织和个人

**格式要求：**
| 元素 | 格式 |
|------|------|
| 标题 | 三号黑体，段前0.5行，段后0.5行 |
| 正文 | 宋体小四号，首行缩进，1.25行距 |

**注意：** 字数以不超过一页纸为宜

## 四、参考文献著录格式

### 4.1 期刊文献 [J]

**格式：**
```
[序号] 主要作者. 文献名[J]. 出版物题名, 年, 卷(期): 页码.
```

**作者规则：**
- 作者不超过3位：全部列出
- 超过3位：前三位作者 + "等"或"et al"

**示例：**
```
[1] 向树红, 商圣飞, 沈自才, 等. 高超声速气膜冷却技术研究进展及发展方向[J]. 宇航材料工艺, 2020, 50(3): 1-10.
[2] Che Y, Wang Q, Xie Z, et al. High-quality images and data augmentation[J]. Computers and Electronics in Agriculture, 2023, 212: 1-11.
```

**预印本格式：**
```
[序号] 主要作者. 文献名[J/OL]. 出版物名: doi号或网址, 年.
```

### 4.2 会议文集 [C]

**格式：**
```
[序号] 姓名. 会议论文题目[C]. 会议名称或出版社, 年限.
```

### 4.3 学位论文 [D]

**格式：**
```
[序号] 姓名. 论文名[D]. 机构地: 机构名, 年份.
```

### 4.4 报告文献 [R]

**格式：**
```
[序号] 姓名. 报告题目[R]. 报告地: 机构名, 年份.
```

### 4.5 普通图书文献 [M]

**格式：**
```
[序号] 姓名. 图书名[M]. 出版地: 出版社, 年份.
```

### 4.6 技术标准 [S]

**格式：**
```
[序号] 主要责任者. 标准名 标准编号[S]. 出版地: 出版者, 出版年.
```

### 4.7 专利文献 [P]

**格式：**
```
[序号] 专利申请者. 专利题名: 专利号[P]. 国别, 公告日期.
```

### 4.8 电子文献 [EB/OL]

**格式：**
```
[序号] 主要责任者. 题名[文献类型/载体标识]. 出版地: 出版者, 时间 [引用日期]. 访问路径.
```

**载体标识：**
- [DB/OL]：联机网络数据库
- [M/CD]：光盘图书
- [J/OL]：联机网络期刊
- [EB/OL]：联机网络电子公告
- [Z/OL]：其他网络形式（微信公众号、微博等）

## 五、质量要求

1. 论文应根据导师、评阅人和答辩委员的意见进行认真修改
2. 文中的错别字率不得超过**1‰**
3. 必须图表清晰（非复印件）
4. 确保质量

---

## 六、格式检查清单

### 6.1 整体检查

- [ ] 论文字数是否达到3万字以上
- [ ] 各部分顺序是否正确
- [ ] 页边距设置是否正确（上左3cm，下右2.5cm）
- [ ] 页眉内容是否正确（偶数页题目，奇数页章节）
- [ ] 页码格式是否正确（摘要目录罗马数字，正文阿拉伯数字）

### 6.2 摘要检查

- [ ] 中文摘要约500字
- [ ] 关键词3-5个
- [ ] 英文摘要与中文对应
- [ ] 格式符合要求

### 6.3 正文检查

- [ ] 一级标题：黑体三号
- [ ] 二级标题：黑体小三号
- [ ] 三级标题：宋体四号
- [ ] 正文：宋体小四号，首行缩进，1.25行距
- [ ] 图表编号规范（阿拉伯数字分级）
- [ ] 图在下方标注，表在上方标注
- [ ] 表格使用三线表
- [ ] 公式编号在右边

### 6.4 参考文献检查

- [ ] 使用顺序编码制 [1], [2], [3]...
- [ ] 引用标注使用上角标
- [ ] 各类文献格式正确
- [ ] 作者超过3人时使用"等"或"et al"

### 6.5 其他检查

- [ ] 个人简介和导师简介分页
- [ ] 致谢不超过一页
- [ ] 错别字率低于1‰
- [ ] 图表清晰
