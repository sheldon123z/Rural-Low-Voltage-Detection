## 四、图表规范

### 4.1 格式要求

| 属性 | 要求 |
|------|------|
| 分辨率 | ≥300 DPI |
| 格式 | PDF (矢量图优先) 或 PNG |
| 字体 | 宋体/SimSun (中文)，Times New Roman (英文) |
| 字号 | 标题 10.5pt，标注 9pt |

### 4.2 配色方案

| 元素 | 推荐颜色 |
|------|----------|
| TimesNet 系列 | 蓝色系 (#1f77b4, #aec7e8) |
| Transformer 系列 | 橙色系 (#ff7f0e, #ffbb78) |
| 轻量级模型 | 绿色系 (#2ca02c, #98df8a) |
| 异常标注 | 红色 (#d62728) |
| 正常区域 | 灰色 (#7f7f7f) |

### 4.3 命名规范

- 中文命名，下划线分隔
- 格式: `[章节]_[内容描述].[格式]`
- 示例: `chap4_F1分数对比.png`

## 六、图表引用格式 (LaTeX)

### 6.1 插入图片

```latex
\begin{figure}[htbp]
    \centering
    \includegraphics[width=0.8\textwidth]{figures/chap4/chap4_F1分数对比.png}
    \caption{PSM数据集各模型F1分数对比}
    \label{fig:f1_comparison}
\end{figure}
```

### 6.2 子图排列

```latex
\begin{figure}[htbp]
    \centering
    \subfigure[训练曲线]{
        \includegraphics[width=0.45\textwidth]{figures/chap4/chap4_训练曲线对比.png}
        \label{fig:train_curve}
    }
    \subfigure[F1分数]{
        \includegraphics[width=0.45\textwidth]{figures/chap4/chap4_F1分数对比.png}
        \label{fig:f1_score}
    }
    \caption{PSM数据集实验结果}
    \label{fig:psm_results}
\end{figure}
```
