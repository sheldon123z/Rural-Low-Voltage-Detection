## 五、图表设计规范

### 5.1 配色方案

论文图表使用统一配色方案：

| 颜色代码 | 用途 |
|---------|------|
| `#2E86AB` | 主色（TPATimesNet） |
| `#A23B72` | 次色（TimesNet） |
| `#F18F01` | 辅色（MTSTimesNet） |
| `#C73E1D` | 强调色（VoltageTimesNet） |

### 5.2 字体规范

- 标题：14pt，加粗
- 轴标签：12pt
- 刻度标签：10pt
- 图例：10pt

### 5.3 输出格式

- **论文使用**: PDF 矢量格式
- **备份/预览**: PNG 300 DPI
