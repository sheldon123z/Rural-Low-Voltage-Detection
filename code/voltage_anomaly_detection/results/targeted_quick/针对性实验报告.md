## 4. 结论与建议

### 4.1 模型选型建议

| 应用场景 | 推荐模型 | 说明 |
|----------|----------|------|
| 常规电压监测 | TimesNet | 通用性强，基准性能可靠 |
| 周期性负荷明显 | VoltageTimesNet | 预设电网周期，针对性强 |
| 三相系统监测 | TPATimesNet | 三相注意力，捕捉不平衡 |
| 复杂混合异常 | MTSTimesNet | 多尺度建模，综合能力强 |
| 高噪声环境 | HybridTimesNet | 置信度融合，鲁棒性强 |
