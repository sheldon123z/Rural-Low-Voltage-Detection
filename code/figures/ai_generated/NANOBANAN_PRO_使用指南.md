# Nano Banana Pro 论文图表生成指南

> 农村低压配电网电压异常检测论文 - 学术图表生成

## 📋 使用说明

由于 OpenRouter 的图像生成模型在中国大陆地区受限，请直接访问 **Nano Banana Pro 网页版** 手动生成图表。

### 步骤

1. 访问 [Nano Banana Pro](https://nanobanana.ai/) 或 [Google AI Studio](https://aistudio.google.com/)
2. 选择 **Thought Mode** 或 **Pro Mode**（更稳定的输出）
3. 复制下方对应图表的提示词
4. 粘贴到输入框并生成
5. 下载生成的图片（选择最高分辨率）
6. 必要时在 Figma/PowerPoint 中进行后期处理

### ⚠️ 重要要求

| 要求 | 规格 |
|------|------|
| **文字语言** | 仅中文，禁止英文 |
| **中文字体** | 宋体（SimSun） |
| **字号** | ≥12号，确保清晰可读 |
| **分辨率** | 4K 高清，300 DPI |
| **边缘** | 裁剪紧凑，减少空白 |
| **尺寸** | 适合 A4 论文（宽度约 15cm） |
| **配色** | 学术蓝 #0076A8，深灰 #333333，白底 |

## 💡 使用技巧

### 1. 模式选择
- **推荐使用 Thought Mode**：输出更稳定、更符合描述
- 如果生成效果不好，尝试切换到 Pro Mode

### 2. 迭代优化
如果第一次生成不理想，可以添加更具体的要求：
- "请确保所有文字使用中文宋体"
- "图片边缘要更紧凑，减少空白"
- "颜色使用更深的蓝色"

### 3. 后期处理
生成后可在以下工具中微调：
- **Figma**：调整布局、添加精确标注
- **PowerPoint**：快速添加中文文字
- **Photoshop**：调整颜色、裁剪边缘

### 4. 分辨率
- 下载时选择最高分辨率
- 如果需要更高分辨率，可以使用 AI 图像放大工具

### 5. 常见问题

**Q: 图像中出现了英文怎么办？**
A: 在提示词末尾强调"禁止出现任何英文，所有文字必须是简体中文"

**Q: 颜色不够准确？**
A: 明确指定 hex 颜色代码，如"学术蓝色 (#0076A8)"

**Q: 图片边缘空白太多？**
A: 添加"图片边缘紧凑，裁剪掉多余空白"

**Q: 字体不是宋体？**
A: 这是 AI 的局限性，可在后期处理中替换文字
